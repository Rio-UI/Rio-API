# 接入 OpenAI Translator

Openai Translator 是一款跨平台的划词翻译应用和浏览器插件

‍

# 下载App，进入设置页

从官网下载 App: https://github.com/openai-translator/openai-translator

打开 App，进入设置页

​​

# 配置API

在设置页添加 Rio API 服务，配置 API 密钥 (Key)、API 地址 (End point) 和模型 (Model ID)

![image](https://pic.riiio.top/image-20250426182917-tljdatq.png)​​​

保存，开始使用​​
